# UPPCS Short Notes

Comprehensive study materials for UPPCS (Uttar Pradesh Public Service Commission) examination.

## Subjects Covered

- **Ancient India** - Stone Age, Indus Valley, Vedic Age, Mauryan Empire, Gupta Period
- **Medieval India** - Delhi Sultanate, Mughal Empire, Regional Kingdoms
- **Modern India** - British Rule, Freedom Struggle, National Movement
- **Art & Culture** - Indian Art, Architecture, Literature
- **Economics** - Money & Banking, Budget & Taxation, Balance of Payments, Sectors & GDP, Infrastructure, Human Resource Development

## Features

- Dark mode support
- Responsive design
- Comprehensive notes with PYQ (Previous Year Questions)
- Easy navigation between topics
